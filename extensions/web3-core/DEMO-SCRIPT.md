**脚本**:

```

选择关联的租约ID，

"GPU性能比广告宣传的低50%"

提交后，系统会自动创建案件号，
并通知相关方。

预计响应时间是24-48小时。
```

## 📹 录制技术要求

### 环境准备

```bash
# 1. 确保服务运行
npm start

# 2. 准备测试数据
npm run seed:demo-data

# 3. 清除浏览器缓存
# Ctrl+Shift+Del

# 4. 设置浏览器窗口
# 尺寸: 1920x1080
# 缩放: 100%
```

### 录制设置

| 项目       | 设置           |
| ---------- | -------------- |
| **分辨率** | 1920x1080      |
| **帧率**   | 30 fps         |
| **码率**   | 5-8 Mbps       |
| **音频**   | 48kHz, 128kbps |
| **格式**   | MP4 (H.264)    |
| **时长**   | 15分钟         |

### 录制工具推荐

- **macOS**: QuickTime / ScreenFlow / Camtasia
- **Windows**: OBS Studio / Camtasia
- **Linux**: SimpleScreenRecorder / OBS Studio

## 🎤 旁白技巧

### 语速控制

- 正常语速：150-180字/分钟
- 关键点时放慢语速
- 演示操作时适当停顿

### 语气

- 自信、友好
- 避免"嗯"、"啊"等口头禅
- 适当的热情，但不过度

### 声音

- 清晰、洪亮
- 使用麦克风录制
- 降噪处理

## ✂️ 后期处理

### 剪辑要点

1. **移除冗余**
   - 加载等待时间
   - 失误操作
   - 过长的停顿

2. **添加字幕**
   - 中文字幕（必须）
   - 英文字幕（可选）
   - 突出关键信息

3. **添加标注**
   - 圈出关键元素
   - 箭头指示
   - 文字说明

4. **片头片尾**
   - 片头：项目Logo + 标题（3秒）
   - 片尾：联系方式 + 致谢（3秒）

5. **背景音乐**（可选）
   - 轻音乐，音量低
   - 不要盖过旁白
   - 使用无版权音乐
