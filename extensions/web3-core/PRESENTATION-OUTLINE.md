## 🎨 设计规范

### 配色方案 - Midnight Executive

| 用途     | 颜色代码  | 颜色名 |
| -------- | --------- | ------ |
| 主色     | `#1E2761` | 深蓝色 |
| 次色     | `#CADCFC` | 冰蓝色 |
| 强调色   | `#FFFFFF` | 白色   |
| 背景暗色 | `#0D1117` | 深灰色 |
| 成功色   | `#10B981` | 绿色   |
| 警告色   | `#F59E0B` | 黄色   |
| 错误色   | `#EF4444` | 红色   |

### 字体规范

| 元素       | 字号    | 粗细    |
| ---------- | ------- | ------- |
| 幻灯片标题 | 36-48pt | Bold    |
| 章节标题   | 20-24pt | Bold    |
| 正文文本   | 14-16pt | Regular |
| 说明文字   | 10-12pt | Regular |

### 布局原则

- **留白**: 至少0.5英寸边距
- **对齐**: 左对齐正文，居中标题
- **一致性**: 所有slides使用统一的间距和布局
- **视觉元素**: 每张slides至少包含一个图标、图表或图片

## 📝 制作说明

### 工具选择

**推荐工具**:

- Microsoft PowerPoint
- Google Slides
- Keynote (macOS)
- Canva (在线)

### 制作步骤

1. **选择模板**
   - 使用16:9宽屏比例
   - 选择专业的深色主题模板

2. **应用配色**
   - 将模板配色替换为Midnight Executive配色
   - 确保文字对比度足够

3. **添加内容**
   - 按照大纲逐页添加内容
   - 使用图标增强视觉效果

4. **插入图表**
   - 使用PowerPoint内置图表功能
   - 或截图Dashboard中的图表

5. **审查优化**
   - 检查拼写和语法
   - 确保所有页面风格一致
   - 预览全屏效果

### 输出格式

- **演示用**: PPTX格式
- **分享用**: PDF格式
- **在线用**: 上传到SlideShare或Google Slides
