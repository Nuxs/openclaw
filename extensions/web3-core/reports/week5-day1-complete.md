## 💡 使用建议

### 如何使用演示脚本

1. **录制视频使用**

   ```bash
   # 1. 启动Dashboard
   npm start

   # 2. 准备测试数据
   npm run seed:demo-data

   # 3. 按照脚本录制
   # 使用OBS Studio或其他录屏软件
   ```

2. **现场演示使用**
   - 打印脚本作为提示卡
   - 熟记关键操作步骤
   - 预演至少2次

3. **培训材料使用**
   - 作为新用户培训教程
   - 功能介绍参考
   - 操作手册

### 如何制作PPT

1. **使用PowerPoint**
   - 打开PowerPoint
   - 创建空白演示文稿
   - 按照PRESENTATION-OUTLINE.md逐页制作

2. **使用Google Slides**
   - 访问slides.google.com
   - 新建演示文稿
   - 应用配色方案
   - 添加内容

3. **使用Canva**
   - 访问canva.com
   - 选择演示文稿模板
   - 自定义配色
   - 导出为PPTX或PDF

### 录制视频建议

**工具选择**:

- **Windows**: OBS Studio (免费)
- **macOS**: QuickTime / ScreenFlow
- **Linux**: SimpleScreenRecorder

**录制设置**:

```
分辨率: 1920x1080
帧率:   30fps
码率:   5-8 Mbps
格式:   MP4 (H.264)
```

**后期处理**:

- 使用DaVinci Resolve (免费) 或 Adobe Premiere
- 添加字幕
- 添加标注
- 去除冗余部分
